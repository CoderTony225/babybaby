## 🚀 快速启动

### 方法 1：使用启动脚本（最简单）

2. 等待看到 `Started BasePlatformApplication` 提示
3. 打开浏览器访问：**http://localhost:8080**

```powershell

```


## ⚠️ 常见问题

### 问题 1：后端启动失败 - "拒绝访问"

```
ERROR: target\classes\application.yml (拒绝访问)
```

**解决：**
```powershell
cd c:\Users\Tony0\Desktop\cupro\baseplatform
Remove-Item -Recurse -Force target -ErrorAction SilentlyContinue
.\mvnw.cmd clean spring-boot:run
```

### 问题 2：端口被占用

```
Port 8080 was already in use
```

**解决：**
```powershell
# 查找并杀死占用 8080 的进程
netstat -ano | findstr :8080
taskkill /F /PID <进程ID>
```

## 🛠️ 停止服务

- 在运行窗口按 `Ctrl + C`
- 或执行：`taskkill /F /IM java.exe`

- **连接池**: Druid 1.2.16
