## 概述

- **交互形态**：首期以 **Thymeleaf 服务端渲染**为主，下列路径为 **页面与表单提交接口**（非 JSON REST）。
- **认证方式**：**Apache Shiro 会话**（登录后 Cookie 会话）；未登录访问受保护路径将跳转登录页。
- **内容类型**：表单提交多为 `application/x-www-form-urlencoded`；个别场景可使用 `multipart/form-data`（本期附件上传排除）。
- **字符编码**：UTF-8

## 接口详情

### GET /uavs

**描述**：分页查询无人机列表。

**查询参数**：

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| keyword | String | 否 | 关键字，匹配机身编号、型号、备注 |
| page | int | 否 | 页码，从 1 开始 |
| size | int | 否 | 每页条数 |

**响应**：HTML 页面（200 OK）。

---

### GET /uavs/new

**描述**：展示新建表单。

**响应**：HTML 页面（200 OK）。

---

### POST /uavs

**描述**：创建无人机档案；服务端触发 AI 生成扩展属性（失败按设计降级）。

**请求体（表单字段）**：

| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| frameSn | String | 是 | 机身编号，唯一 |
| modelName | String | 是 | 型号 |
| notes | String | 否 | 备注 |

**成功响应**：302 重定向至详情或列表（以实现为准），Flash 成功消息。

**失败响应**：200 + 表单页重新渲染并展示校验或业务错误。

---

### GET /uavs/{id}

**描述**：无人机详情。

**路径参数**：`id` — 主键。

**响应**：HTML 页面（200 OK）；不存在时 404 或错误页。

---

### GET /uavs/{id}/edit

**描述**：编辑表单。

**路径参数**：`id` — 主键。

**响应**：HTML 页面（200 OK）。

---

### POST /uavs/{id}

**描述**：提交更新。

**路径参数**：`id` — 主键。

**请求体（表单字段）**：与创建类似；是否允许修改 `frameSn` 以实现为准（建议禁止修改唯一编号）。

**成功响应**：302 重定向 + Flash 消息。

---

### POST /uavs/{id}/delete

**描述**：删除记录。

**路径参数**：`id` — 主键。

**成功响应**：302 重定向至列表 + Flash 消息。

---

## 错误与提示约定

- **校验错误**：表单字段级错误信息绑定至 Thymeleaf。
- **业务错误**：如机身编号重复，展示全局错误消息。
- **AI 降级**：保存成功后页面提示 AI 未生成或部分失败（与设计中的 `aiGenerationStatus` 一致）。
